Web Applications
A collection of web application labs, assignments, and practice projects focused on 
building modern full-stack apps. This repo covers both small, focused exercises and 
larger multi-file projects that combine UI work, backend APIs, and database features.

Tech Stack:
- MongoDB
- Express
- React
- Node.js
- HTML, CSS, JavaScript

Repository Structure:
Projects are organized into folders, and each folder contains the code and an MP4 file demonstrating the web page.
